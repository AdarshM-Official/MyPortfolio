# 🌐 My Portfolio — Built with Django

Welcome to my personal portfolio project!  
This is a dynamic web application built using Django to showcase my work, skills, and projects in a structured and scalable way.

---

## ✨ What makes this different?

Unlike a static portfolio, this project is:
- 🔄 **Dynamic** — Content can be updated easily from the backend
- 🧠 **Scalable** — Designed with modular Django apps
- ⚙️ **Extendable** — Ready for features like authentication, blog, or APIs
- 💻 **Developer-focused** — Clean structure and best practices

---

## 🚀 Features

- 🧩 Modular Django architecture (`core` app)
- 🗂️ Organized project structure for scalability
- ⚡ Fast development with Django built-in tools
- 🛢️ SQLite database (easy to upgrade later)
- 🔐 Ready for authentication & user features (future scope)

---

| Technology | Purpose |
